# Now Playing

What is Injoon listening to right now — plus recent photos.

Data is refreshed on a schedule by GitHub Actions and served as static JSON.

## API

```
https://raw.githubusercontent.com/anaclumos/now-playing/main/now-playing.json
https://raw.githubusercontent.com/anaclumos/now-playing/main/photos.json
```
